# Customer Churn Prediction using Machine Learning

## Overview
Customer churn prediction helps businesses identify customers who are likely to stop using their services. This project applies multiple machine learning algorithms to predict customer churn and identify key factors influencing customer retention.

## Problem Statement
The objective is to build a classification model that predicts whether a customer will churn based on demographic, billing, and service usage information.

## Dataset
- Telco Customer Churn Dataset
- Features include:
  - Customer Demographics
  - Contract Details
  - Monthly Charges
  - Total Charges
  - Tenure
  - Payment Method
  - Internet Services
  - Churn Status
